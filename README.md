# Commercial-Performance-Profitability-Analytics-Dashboard

## 📊 Project Overview
This Business Intelligence project focuses on evaluating the 2024 sales performance and profitability for **VisioMetrics Analytics**, a technology importer in Costa Rica. The analysis transforms raw sales data into a strategic decision-making tool to identify operational deficits and high-COGS (Cost of Goods Sold) patterns.

## 🛠️ Technical Stack
- **Tool:** Microsoft Excel (Advanced)
- **ETL Process:** Power Query for data cleaning, standardization, and transformation.
- **Modeling:** Pivot Tables and calculated fields for margin analysis.
- **Visualization:** Interactive Dashboards with dynamic slicing and drill-down capabilities.

## 📈 Key Analytical Findings
- **Cost Structure:** COGS accounts for 73% of total sales, resulting in an 8% net loss operational deficit.
- **Category Performance:** "Sensores" and "Seguridad" are top profit drivers, while "Redes" and "Asistentes" concentrate structural losses.
- **Geographic Insights:** San José is the primary hub but also records the highest absolute net loss, suggesting a need for product-mix optimization.
- **Sales Team Impact:** Identified a significant profitability gap between top-volume sellers and high-margin contributors.
